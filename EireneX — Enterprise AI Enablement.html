Common Questions

Practical answers
to real concerns

Do we need GPU infrastructure to use EireneX?

+

No. For basic and many intermediate setups, EireneX can run without dedicated GPU hardware. A capable MacBook or standard server is sufficient to get started. If your workloads grow to require GPU-accelerated local inference, we can support that path — but it is never a starting requirement.

Can this run on-premises?

+

Yes. On-prem deployment is supported across all three EireneX solutions. Model Selector specifically includes an on-prem model option for fully internal, air-gapped chat environments. If your security posture requires data never to leave your infrastructure, we configure accordingly.

Is our data secure?

+

Security is a design consideration, not a feature layer. The on-prem configuration ensures sensitive data doesn't flow through external APIs unless you explicitly choose that. Token management, prompt routing, and model selection are all configurable within your security boundaries. We do not store your data.

What models do you support?

+

EireneX connects to 119+ models across open-source, proprietary, and locally-deployable options. These include models suited for code generation, document work, structured output, summarisation, and more. Model Selector helps route each prompt to the appropriate model rather than defaulting to one expensive option for all tasks.

Can we start with just one solution?

+

Yes, and that's how most organizations begin. The $1,000 setup includes the base environment and whichever solution is most relevant to start. The other solutions can be added progressively — same infrastructure, no additional setup cost.

Is this only for developer teams?

+

No. Development Workbench serves engineering and delivery teams. MarketingFlow serves marketing functions. Model Selector serves any team that needs secure AI chat. EireneX is designed to enable multiple departments from one foundation — not just engineering.

How long does setup take?

+

Setup timelines depend on your environment and the number of solutions being deployed. A single-solution setup on a standard environment typically completes within a few days. We provide a documented handover so your team can operate the setup independently after the initial configuration.

What if we already have some AI tooling in place?

+

EireneX is built on open and practical tooling, which means it can often sit alongside or connect to what you already have. During the setup call, we review your current environment and design the configuration around it — we don't require you to replace what works.
